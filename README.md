# MediaStore API
This is the beginning of MediaStore and build in ASP.NET Core Web API. 

## Description
MediaStore is an interface for creating, reading, updating and deleting (CRUD) informasjon about digital media. The first step will include francises, movies and characters. 

At the moment the schemas in Swagger UI is diabled because of the amount of schemas. I hope the documentation for the endpoints cover your needs. If you have wishes or input for the next step, don't hesistate to contact me.

## Getting started
Clone the code to a local directory

Open solution in Visual Studio (eventually Visual Studio Code). The code is build in Visual Studio 2022 and .NET 5.0

Run migrations

Run IIS Express
